# GameClub

A free, community-maintained directory of board game clubs in Yorkshire, featuring an interactive map powered by Leaflet.js and OpenStreetMap.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to view the site.

## Adding a Club

See [CONTRIBUTING.md](CONTRIBUTING.md) for instructions on how to add your club to the directory.

## Tech Stack

- Jekyll (GitHub Pages)
- Leaflet.js + OpenStreetMap
- Vanilla JS
- Jekyll Sass
